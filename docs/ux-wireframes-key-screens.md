## 📐 WIREFRAME KONVENCIÓK

### Méretezési Szabványok
- **Desktop**: 1024px szélesség
- **Tablet**: 768px szélesség  
- **Mobil**: 320px szélesség
- **Gomb méretek**: Minimum 44px (touch-friendly)
- **Margó**: 20px alapértelmezett
- **Grid rendszer**: 12 oszlop desktop, 6 oszlop tablet

### Színkódolás
```
███ = Kötelező elem
▓▓▓ = Interaktív elem
░░░ = Háttér elem
▬▬▬ = Navigációs elem
●●● = Szöveg/label
```

## 🔧 WIREFRAME SPECIFIKÁCIÓK

### Kötelező Elemek Minden Képernyőhöz:
1. **Header navigáció**: Logo + fő navigációs elemek
2. **Breadcrumb**: Aktuális pozíció jelzése
3. **Progress indicator**: Haladás mutatása
4. **Footer**: Copyright + links

### Interakciós Elemek:
- **Gombok**: Minimum 44x44px touch targets
- **Input mezők**: 48px magas, clear labels
- **Navigáció**: Logical tab order
- **Feedback**: Azonnali vizuális visszajelzés

### Accessibility Requirements:
- **Kontraszt**: Minimum 4.5:1 ratio
- **Focus states**: Clear keyboard navigation
- **Screen reader**: ARIA labels
- **Scalable text**: Responsive font sizing
